# game-engine

## What is this game

Work in space as a transporter.

## Features

-   pick up and deliver containers
-   navigate in space

## First time set up

```
npm i -g eslint
npm i -g @stylistic/eslint-plugin-js
npm i -g concurrently
npm i -g reload 
```

## run locally

```
./run.sh
```

## Run linter

```
./lint.sh
```
